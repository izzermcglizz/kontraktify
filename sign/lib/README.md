# Required JavaScript Libraries

Download these files and place them in this folder:

## 1. Supabase JS Client
- **File:** `supabase.js`
- **URL:** https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2/dist/umd/supabase.min.js
- **Size:** ~100KB

## 2. PDF.js (for rendering PDFs)
- **File:** `pdf.min.js`
- **URL:** https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js

- **File:** `pdf.worker.min.js` 
- **URL:** https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js

## 3. PDF-lib (for modifying PDFs)
- **File:** `pdf-lib.min.js`
- **URL:** https://cdn.jsdelivr.net/npm/pdf-lib@1.17.1/dist/pdf-lib.min.js

## 4. Signature Pad (for drawing signatures)
- **File:** `signature_pad.min.js`
- **URL:** https://cdn.jsdelivr.net/npm/signature_pad@4.1.7/dist/signature_pad.umd.min.js

## How to Download:
1. Click each URL above
2. Right-click on the page â†’ "Save As"
3. Save with the filename shown above
4. Place all files in this `/sign/lib/` folder

```html

```
